# temple

**TEMPL**ate **E**ngine. 

A jinja2-like template engine in [rust] inspired by  *[Jinja2Cpp]*.

[Jinja2Cpp]: https://github.com/jinja2cpp/jinja2cpp
[rust]: https://www.rust-lang.org
